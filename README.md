## 📝 Descripción
Aplicación web desarrollada para la gestión integral de una liga de básquetbol. Permite a los administradores llevar el control de equipos, gestionar jugadores, registrar resultados de los partidos y visualizar la tabla de posiciones actualizada de forma dinámica. 

Proyecto desarrollado como Práctica Final para el curso de Ingeniería de Software 2 (IS2).

## 🚀 Tecnologías utilizadas
* **Backend:** Laravel (PHP)
* **Frontend:** Blade, CSS/Tailwind, JavaScript
* **Base de Datos:** MySQL / SQLite
* **Control de Versiones:** Git & GitHub (GitFlow)

## 🛠️ Metodología y Flujo de Trabajo
Este proyecto se desarrolló aplicando metodologías ágiles y buenas prácticas de trabajo colaborativo para simular un entorno real de desarrollo:
* **Gestión de Proyecto:** Tablero Kanban integrado en GitHub Projects, gestionando el Backlog a través de Historias de Usuario estimadas por puntos de esfuerzo.
* **GitFlow:** Uso estricto de ramas para la creación de nuevas funcionalidades (`feature/nombre-de-la-tarea`).
* **Integración Continua:** Revisión de código obligatoria (Code Review) mediante Pull Requests. Ninguna funcionalidad se integró a `main` sin la aprobación previa de al menos un miembro del equipo.

## ⚙️ Requisitos previos
* PHP >= 8.1
* Composer
* Node.js y NPM
* Base de datos MySQL o SQLite
